Ah man, Right so for all artists who never did an inktober challenger before this is basically how it should be done. You start going through the prompt list in order from the number 1 all the way to number 31 (which is how many days there are in october). Each day is labeled with a prompt and the challenge is that you should be doing that certain prompt on that given day. So today is 1, meaning today's prompt is Boyfriend, tomorrow is 2 so tomorrow would be Rivalry (for example 2 people who dislike one another such as Tabi and Bf and etc..) As mentioned before, each prompt is subjective, meaning it's up to you on what it should or might contain, also meaning that it doesn't necessarily have to be Halloween related unless the prompt suggests so.
